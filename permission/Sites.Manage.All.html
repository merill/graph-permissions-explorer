Sites.Manage.All

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the Sites.Manage.All permission.

Allows the application to create or delete document libraries and lists in all site collections on behalf of the signed-in user.

Category Application Delegated
Identifier 0c0bf378-bf22-4481-8f81-9e89a9b4960a 65e50fdc-43b7-4915-933e-e8138f11f40a

Description Allows the app to create or delete document libraries and lists in all site collections without a signed in user. Allows the application to create or delete document libraries and lists in all site collections on behalf of the signed-in user.
