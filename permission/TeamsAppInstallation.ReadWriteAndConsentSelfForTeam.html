TeamsAppInstallation.ReadWriteAndConsentSelfForTeam

Allows a Teams app to read, install, upgrade, and uninstall itself in teams the signed-in user can access, and manage its permission grants for accessing those specific teams' data.

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the TeamsAppInstallation.ReadWriteAndConsentSelfForTeam permission.

Granting this permission allows the calling application to access the following information from your tenant.

  • teamsApp
  • teamsAppInstallation
  • teamsAppPermissionSet
  • teamworkBot
