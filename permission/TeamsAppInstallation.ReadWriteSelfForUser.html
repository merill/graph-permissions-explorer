TeamsAppInstallation.ReadWriteSelfForUser

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the TeamsAppInstallation.ReadWriteSelfForUser permission.

Allows a Teams app to read, install, upgrade, and uninstall itself for the signed-in user.
