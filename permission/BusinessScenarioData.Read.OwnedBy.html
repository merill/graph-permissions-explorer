BusinessScenarioData.Read.OwnedBy

Allows the app to read all data associated with the business scenarios it owns. Data access will be attributed to the signed-in user.

For an app to access data in Microsoft Graph, the user or administrator must grant it the permissions it needs. This article lists all the Microsoft Graph APIs and your tenant data that can be accessed by the application (vendor/developer) if you consent to the BusinessScenarioData.Read.OwnedBy permission.
